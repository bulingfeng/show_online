# 关于价值投资的讨论

价值投资的定义：

```
维基百科解释：
价值投资是投资策略的其中一种，由本杰明·葛拉汉和大卫·多德所提出。目的就是为了去寻找并投资于一些股价被低估了的股票，然后投资他们。
个人认为：
价值投资就是站在长期的未来的某个时间点，我们曾经持有或者投资的标的已经获取了丰厚的回报。
```

价值投资真的可信吗？

```
价值投资都存在一定的主观性，比如当年最火的乐视和暴风，在那个时间点如果有人买了乐视和暴风的股票，并且说自己是价值投资，我相信大家都不会反对吧。因为它的股票实现了暴涨，市值翻了几倍，这时候乐视不是大家眼中的骗子，而是下一个阿里或者腾讯。
2020年5月14日乐视已经被深圳交易所强制退市，市值从1700亿暴跌倒67亿，跌幅达到99%以上。现在我们都骂乐视是骗子，当年呢？或许当年乐视就是我们心中的最佳价值投资标的。
任何投资标的都离不开大环境的影响，在熊市具有价值投资的股票也会表现的很差劲，在牛市几乎所有股票都在涨，这时候的垃圾股涨幅可能比优质股涨幅更大。所以不管在熊市和牛市寻找价值投资都是一件很困难的事情，熊市的时候整个市场不行，优质股也会表现出垃圾股特性，等到了牛市垃圾股的涨幅远远超过了优质股，这时候优质股又是被低估了。是不是很绝望，难道真的没有方法来筛选出优质股了吗？
我来说下我关于优质的价值投资标的该有的特性
1、好的投资标的应该是在长期看来具有高回报率，而不是短期暴涨，随后暴跌，直至最后破产。
2、好的投资标的应该是反脆弱的，在经过市场的多次洗礼后，不但没有阵亡，反而更强大了。
3、好的投资标的一定是能够经受的住时间的考验，时间是一个好的过滤器，活的久的企业一定是被社会所需要的。
4、好的投资标的大多数具有垄断性。
当然会有同学会问：我们在不能在大量的现在还默默无闻的公司中寻找出下一个阿里或者腾讯呢？
答案是概率为0.至少对于我这个普通人来说概率为0.首先自己的认知能力，人际关系，财务都决定了我自己不能找到这家公司，而且即使找到了自己也没有足够的资金来投资人家。（全世界有多少精英在做这件事呢，作为普通人，我们根本没有胜算的机会）
最后结尾，说出我当韭菜的总结出来的经验：
1、投资其实就是变相的赌博和下注，不要总想着一夜暴富，不赔钱活下去才是最基本的也是最重要的。
2、投资不能保证百分之百成功，我们只有提高投资的成功率。所以我只投行业里面的霸者。
3、别只看重标的的价格，遵“循赢者通吃”，只投资垄断行业。（其实和第2条类似）
我说的以上观点纯属个人理解和思考，不作为任何投资建议。
```
