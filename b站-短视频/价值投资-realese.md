# 关于价值投资的讨论

价值投资的定义：

```
维基百科解释：
价值投资是投资策略的其中一种。目的就是为了去寻找并投资于一些股价被低估了的股票，然后投资它们。
个人认为：
价值投资就是能够在长期的未来能给自己带来丰厚的回报。
```

价值投资真的可信吗？

```
价值投资在任何时候都存在一定的主观性。由于任何投资标的都离不开市场的影响，在熊市即使是优质标的也很难逃脱暴跌的命运，在牛市垃圾标的的涨幅很可能是优质投资标的涨幅的好多倍。所以造成了在熊市里看着任何标的都是毫无前途，在牛市任何投资标的都在暴涨，而其中涨的最猛的反而有可能是最劣质的标的。还有一种投资标的，在初期不但没有被看做优质的投资标的，甚至连劣质的投资标的都算不上，那时候大家都喜欢称它为骗局。直到多年以后大家才发现原来人家是黄金标的。所以找出来一个优质投资标的是很困难的，很主观的。但与此同时找到最优质的的投资标的又是极其容易的。
因为优质的投资标的都具有如下特征：
1、好的投资标的是一个值得长期持有的，而且未来的价值会比现在更高。所以如果你看到有人喊，如果你错过这个投资机会你就失去了一次财富自由的机会，这类人一定是骗子。
2、好的投资标的应该具有反脆弱性，也就是说经历过多次低谷后，反而活的越来越好了。
3、好的投资标的大多数具有垄断性，比如可口可乐，apple，谷歌，微软，Facebook，特斯拉等等。
好了到此，我的分享就结束了，但是我要提醒大家的是，以上都是我自己的思考和认知，不作为任何投资建议。另外，提醒大家，如果有人教你如何实现财富自由，请大家一定要警惕起来，因为对方百分之百是骗子，没有例外。
最后的最后，我知道很多小伙伴还在买一些便宜的投资标的，他们期待这些公司是行业中的黑马，或许未来的阿里或者腾讯，到那个时候自己就可以坐拥白富美了。这个想法到底靠不靠谱呢？欢迎大家留言，把自己的想法和感受写到评论区，大家一起讨论进步。
```
